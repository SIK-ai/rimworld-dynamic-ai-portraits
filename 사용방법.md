# 🎨 Dynamic AI Portraits — 사용방법

콜로니스트의 **현재 상태**(복장·무기·부상·정신 상태·종족·이념·임신·중독 등)를 실시간으로 반영한 **AI 초상화**와 **4초 루프 영상**을 생성해, 인스펙트 창 위에 깔끔한 투명 오버레이로 띄워 주는 RimWorld 1.5 / 1.6 모드입니다.

---

## 📦 설치

1. [릴리스](https://github.com/SIK-ai/rimworld-dynamic-ai-portraits/releases)에서 zip을 받아 압축을 풀고, 폴더 이름을 `AIPortraits`로 바꿉니다.
2. `AIPortraits/` 폴더를 RimWorld의 `Mods` 폴더에 넣습니다.
3. 모드 목록에서 **Harmony를 먼저** 켜고, 그다음 **Dynamic AI Portraits**를 켭니다.
4. **옵션 → 모드 설정 → Dynamic AI Portraits**에서 제공자(백엔드)를 고릅니다.

> ⚠️ **Harmony 필수** — [Steam 워크샵](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077)에서 받으세요.

## 🔌 백엔드(이미지 생성 제공자) 고르기

> 💡 **처음이라면 무료부터.** 그리고 유료 제공자를 골랐는데 키가 없거나 요청이 실패하면, 모드가 **자동으로 무료 Pollinations로 폴백**하므로 초상화는 항상 나옵니다.

| 제공자 | 비용 | 속도 | 비고 |
|---|---|---|---|
| 🆓 **Pollinations** | 완전 무료, 키 불필요 | ~50초 | 가장 부담 없이 시작 |
| ☁ **Cloudflare Workers AI** | **하루 1만 건 무료**, 이후 ~$0.0005/장 | ~2–4초 | **대부분의 유저에게 추천** |
| 💎 **Google Imagen 4** | ~$0.02/장 | ~3초 | 최고 품질, 네이티브 투명 PNG |
| ⚡ DeepInfra | ~$0.0005/장 | ~2–5초 | 저렴한 유료 |
| 🤗 HuggingFace | 소액 무료 후 종량제 | 가변 | |
| 🖥 로컬 GPU | 무료 (내 PC) | GPU에 따라 | A1111 / Forge / ComfyUI |

**Cloudflare 설정:** [dash.cloudflare.com](https://dash.cloudflare.com)에서 **Account ID**와 `Workers AI Read` 권한의 **API 토큰**을 만든 뒤, API 키 칸에 `account_id:token` 형식(콜론 하나로 구분)으로 붙여 넣습니다.

## ▶ 기본 사용법

- 콜로니스트를 **선택**하고 인스펙트 창의 **탭을 모두 닫으면** 초상화 오버레이가 뜹니다.
- 오버레이의 버튼:
  - **↻** — 현재 상태로 다시 생성
  - **P / B / S** — 프레이밍 전환: **P** 인물(버스트업) · **B** 전신 · **S** 스페셜(배경 있는 연출 장면). 각 프레이밍은 따로 저장됩니다.
  - **V** — 비디오 모드 토글 (아래 참고)
- 자동 재생성은 없습니다. ↻를 누르기 전까지 초상화는 그대로 유지됩니다.

### 🎥 영상 루프 (Google Veo 3.1 Lite)
**V**를 켜면 정지 초상화를 **4초 루프 영상**으로 만들어 줍니다(Google API 키 필요). 생성 중에는 "Making alive..."가 표시되고, 완료되면 오버레이와 갤러리에서 매끄럽게 반복 재생됩니다.

## 🧠 (선택) LLM 프롬프트 엔지니어링

API 설정 → **Prompt Generation**에서 `Gemini Flash Lite` 또는 `Gemma 4`(26B/31B)를 고르면, 폰 데이터를 창의적인 이미지 프롬프트로 다듬어 줍니다. 무료 Google AI Studio 키로 동작하며, 실패 시 내장 템플릿으로 자동 폴백합니다.

---

## 🪄 배경 제거 & 저장

- **배경 제거**: 인물/전신 프레이밍은 로컬 **u2netp(오프라인 신경망)** 으로 배경을 투명하게 만듭니다(선택적으로 클라우드 Cloudflare Bria 사용). **스페셜(S)** 프레이밍은 연출 배경을 그대로 유지합니다.
- **저장 위치**: `Documents/RimWorld Portraits/<PawnID>_<이름>/` 폴더에 폰별로 정리되어 저장됩니다.

## 🔐 개인정보

API 키 입력 칸은 화면에서 `*`로 가려지고, 로그에 남는 키·토큰·계정 ID는 자동으로 삭제(sanitize)되어 `Player.log`를 공유해도 노출되지 않습니다.
