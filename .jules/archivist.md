## 2024-05-28 - Dictionary Scribing and Null Checking
**Learning:** When scribing dictionaries using `Scribe_Collections.Look`, we must provide explicit working lists (e.g., `ref myKeys`, `ref myValues`) to ensure keys and values can be correctly serialized and deserialized. Furthermore, assigning default instances to these collections (i.e. `if (myDict == null) myDict = new Dictionary...`) should be done inside `if (Scribe.mode == LoadSaveMode.PostLoadInit)` rather than directly in the general `ExposeData()` flow to avoid overwriting or interfering with the data loading lifecycle.
**Action:** Always declare explicit working lists (`List<TKey>`, `List<TValue>`) for any dictionary serialized via `Scribe_Collections.Look` and wrap null initializations in `LoadSaveMode.PostLoadInit`.
